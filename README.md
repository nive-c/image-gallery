# 🖼️ Light Image Gallery

An interactive image gallery web app built with HTML, CSS, and JavaScript.
Filter images by category and view them in a full-screen lightbox.

## ✨ Features
- Category filters — All, Abstract, Nature, Urban
- Lightbox preview on image click
- Keyboard navigation — Arrow keys to browse, Escape to close
- Responsive grid layout

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🚀 How to Run
Just open `index.html` in your browser. No installation needed.

## 💼 What This Shows
- DOM manipulation and event handling
- CSS Grid for responsive layouts
- Dynamic filtering logic in JavaScript
- Lightbox UI pattern from scratch
